# Function: stringifyWithFunctions()

Function to stringify an Object. This Function will stringify Functions as well.

```ts
stringifyWithFunctions(obj: any, ...args: any[]): string
```

## Parameters

| Parameter | Type    | Description |
| :-------- | :------ | :---------- |
| obj       | `any`   | The Object. |
| ...args   | `any`[] | -           |

## Returns

`string`
