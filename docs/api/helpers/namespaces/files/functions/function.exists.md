# Function: exists()

```ts
exists(path: PathLike): Promise<boolean>
```

## Parameters

| Parameter | Type       | Description                                                                                                                |
| :-------- | :--------- | :------------------------------------------------------------------------------------------------------------------------- |
| path      | `PathLike` | A path to a file or directory. If a URL is provided, it must use the `file:` protocol.<br />URL support is _experimental_. |

## Returns

`Promise`<`boolean`\>
