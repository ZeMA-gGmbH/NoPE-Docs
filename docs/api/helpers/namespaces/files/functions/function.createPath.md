# Function: createPath()

Function to create a File at a given Path. If the File or the dir doesnt exists it will be created.

```ts
createPath(path: string): Promise<string>
```

## Parameters

| Parameter | Type     | Description    |
| :-------- | :------- | :------------- |
| path      | `string` | Read the File. |

## Returns

`Promise`<`string`\>
