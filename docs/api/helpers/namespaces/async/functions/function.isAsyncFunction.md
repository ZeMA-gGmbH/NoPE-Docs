# Function: isAsyncFunction()

Tests if a Function is async or not.

```ts
isAsyncFunction(func: Function): boolean
```

## Parameters

| Parameter | Type                          | Description           |
| :-------- | :---------------------------- | :-------------------- |
| func      | (...`args`: `any`[]) => `any` | The function to test. |

## Returns

`boolean`

The test result.
