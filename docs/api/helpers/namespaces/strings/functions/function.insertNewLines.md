# Function: insertNewLines()

Helper to insert new lines after a given amount of time.

```ts
insertNewLines(str: string, maxLength: number = 100): string[]
```

## Parameters

| Parameter | Type     | Default value |
| :-------- | :------- | :------------ |
| str       | `string` | undefined     |
| maxLength | `number` | 100           |

## Returns

`string`[]
