# Function: camelize()

Function to Camelize a String

```ts
camelize(str: string, char: string = "_"): string
```

## Parameters

| Parameter | Type     | Default value | Description                      |
| :-------- | :------- | :------------ | :------------------------------- |
| str       | `string` | undefined     | The String,                      |
| char      | `string` | "\_"          | A, used to determine "new words" |

## Returns

`string`
