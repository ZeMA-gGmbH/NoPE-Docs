# Function: insert()

Inserts a String in the String

```ts
insert(
  str: string,
  index: number,
  content: string): string
```

## Parameters

| Parameter | Type     | Description                                |
| :-------- | :------- | :----------------------------------------- |
| str       | `string` | base string                                |
| index     | `number` | index where the content should be inserted |
| content   | `string` | the content to insert                      |

## Returns

`string`
