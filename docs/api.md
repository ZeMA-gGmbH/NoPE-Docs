- For Typescript click [here](https://zema-ggmbh.github.io/NoPE-JS/) to see the current version. To build your own docu:
    1. clone `nope-js` 
        ```bash
        git clone https://zema-ggmbh.github.io/NoPE-JS/
        ```
    2. Install everything:
        ```bash
        npm install
        ```
    3. Build the docs:
        ```bash
        npm run-script doc
        ```
